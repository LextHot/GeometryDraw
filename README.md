# GeometryDraw
Данная программа представляет собой оконное приложение, реализующее работу с геометрическими фигурами. Фигуры - треугольник, прямоугольник, ромб, многоугольник (есть возможность ввода количества углов). Функции - создание фигуры, удаление фигуры, перемещение, поворот, масштабирование, изменение цвета.
